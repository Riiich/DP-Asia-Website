DigitalPersona 企業解決方案的技術

數位人科技企業解決方案將最先進的身分驗證技術，與企業網路環境以及業務應用軟體做緊密地整合。

快速進入與單一登入 Single SingOn

數位人科技一直是使用生物特徵做為單一登入 Single SignOn 技術的領導者。 數位人科技開發的 SignOn 管理工具，能讓IT部門對各種業務應用軟體做事前的設定，消費者因此不需為了使用單一登入 SSO，而費心地自行做各種設定。 

數位人科技的 One Touch SignOn 元件可設定成自動訓練已便了解網頁以及商務應用的登入設計。使用在這個設定時，IT 部門能夠快速地將數位人科技的企業解決方案，安裝到新的各種應用之中。如果是更複雜的網站登入與客製化應用，使用具備充分彈性的手動設定，也能滿足所有的應用需求。 One Touch SignOn 工具可支援以基於 JAVA 及終端機設計的各種應用。

一旦IT部門訓練 One Touch SignOn 以支援某個應用方案的登入方式，接下來就可以藉著 Active Directory 的機制佈署，完整地將此設定配置給分眾或是全部的使用用戶。當使用者下一次使用到這個應用方案時，就可以使用指紋來做登入動做了。

與 Active Directory 的整合

數位人科技的 DigitalPersona Pro 通過微軟 Microsoft 的認證，以擴充 Active Directory schema 的方式，來儲存更多的使用者重要訊息。而且因為使用了 Active Directory 本身的使用者介面，所以並不會有新的操作需要再去學習。IT部門可以使用 Active Directory Group Policy Editior 來建立 GPO，然後根據特定的需求來制定軟體使用方式。

數位人科技 DigitalPersona Pro 的設計是與 Windows 系統的安全服務做緊密的整合，用以保護所有的資料與通訊。因為Windows 系統會將所有運行的事件都會被記錄到 Windows Event Log 事件日誌，所以 DigitalPersona Pro 所產生的相關事件，也可使用一般的入侵偵測或是安全稽核工具來操作分析。

數位人科技 DigitalPersona Pro 利用 Active Directory 以及 DNS 的系統發佈機制服務方式，來滿足企業對未來擴充性的考量。當加入新的 DigitalPersona Pro 驗證伺服器時，服務紀錄就會傳送到提供網路內用戶連線資訊的 DNS 伺服器。如此當擴充系統或是系統發生異常(系統超載或是故障)時，系統管理者就不需去為每一個客戶再更改設定，便能讓所有的用戶都能自動地取得服務更新資訊，轉換到另一個服務的伺服器。

指紋的技術

從1997年以來，數位人科技一直是，在商業與個人用途使用指紋生物特徵技術的領導廠商。數位人科技相信，指紋是強身分認證與易於使用間的完美結合。

數位人科技所引領的生物特徵科技，是由Caltech 以及 MIT 的科學家們所設計，並且在全球已有一億個使用者使用。由於擁有業界領先的準確率與可靠性，數位人科技的生物特徵技術，將安全防護做到了最大化並且擁有最好的使用者體驗。

當使用者將手指放到指紋掃瞄器時，數位人科技的軟體會分析掃瞄所得影像並且取得當中的獨特的特徵點 minutiae，並且轉換成為數學式，指紋模塊，來呈現。 因為經過轉換的過程，所以實際的指紋影像是不會被保存下來，如此也可以進一步地保護使用者生物特徵資料的隱私權。
