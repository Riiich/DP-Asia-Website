U.are.U 4500 指紋鍵盤

把所有優點集合到一個時尚且符合人體工學的鍵盤上.

U.are.U 指紋鍵盤是一個內建指紋掃描器的104個按鍵標準WINDOWS鍵盤. 把所有指紋辨識的好處整合到鍵盤當中. 它是為了下列數位人科技的產品所設計: 企業用戶產品, 以及 軟體開發工具包 SDKs. 這個整合性的鍵盤包含了所有指紋掃描器的特性, 準確的驗證, 高耐用程度還有便利性, 但對一般個人電腦使用者來說, 可以不需要同時連結指紋掃描器與鍵盤兩個USB裝置.

在使用上, 只要很簡單地把手指放在掃描器式窗上, 掃描器便會快速且自動地截取指紋特徵的影像, 接著由數位人科技的IDentity引擎來做驗證.

數位人科技利用先進的光學指紋掃描技術, 來增進指紋影像的品質跟穩定性. U.are.U 4500指紋掃描器與IDentity引擎的結合提供了一套無與倫比的辨識能力來讀取最難辨識的指紋與環境.
