One Touch for Linux 軟體開發工具

數位人科技One Touch for Linux軟體開發工具包(SDK)DK) 給了專業開發者在 Linux 2.4或2.6的核心系統應用上, 對指紋辨識安全系統有更多發揮的空間. One Touch for Linux支援數位人科技U.are.U指紋掃描器跟模組. 開發工具提供一套API來擷取指紋掃描器讀取的影像檔案, 特徵提取, 還有指紋的比對, 都能在ANSI C與C++的編程環境. 同時也包含一個完全分散式數位人科技的及時引擎, 範例程式碼, 還有詳細的使用說明手冊.

產品支援下列Linux 2.4或2.6核心平台:

  • Novel Linux Desktop 9 SP3, 核心版本 2.6.5–7.244-default/smp
  • Novell Linux Point of Service SP2 核心版本 2.6.5–7.201-SLRS
  • Novell Linux Point of Service SP3 核心版本 2.6.5–7.155.29-SLRS
  • SUSE Linux Enterprise Desktop 10 SP1 核心版本 2.6.16.46–0.12-default/smp
  • Slackware 11 核心版本 2.6.18
  • Slackware 11 核心版本 2.4.33.3
  • Slackware 12 核心版本 2.6.21.5
  • Red Hat Linux WS V3 核心版本 2.4.21–50.EL/smp

購買技術支援能夠減輕最終解決方案跟應用的研發負擔.

*核心模式的驅動程式原始碼是提供給開發者以便移植在其他的2.6 Linux上執行. 由於各種版本的2.6 Linux作業系統, 數位人科技沒辦法提供技術支援讓開發者重新編譯驅動程式原始碼.
