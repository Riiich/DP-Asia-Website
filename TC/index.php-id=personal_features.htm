DigitalPersona Personal 可以幫你做些甚麼?

DigitalPersona Personal 是一套使用指紋的生物特徵，來進行身分認證的軟體，以避免冒用身分的問題發生。 有了DigitalPersona Personal，讓使用電腦與網路變得更加安全且容易.

DigitalPersona 有下列的優點 :

取代密碼

使用指紋, 快速且安全地登入電子郵件, 線上金融業務, 以及常用的網站與網路應用的技術, 例如 Facebook, Skype, 或是網路銀行帳號 !!

如果網頁上有出現指紋辨識的圖示, 表示 DigitalPersona Personal 可以加入這個網. 只需要輸入一次帳號與密碼, 再掃描指紋及登入之後, 以後就可以用指紋來登入該網頁了.

 

只需要輸入一次帳號與密碼, 之後即可以用指紋來取代.

下一次登入網站時, 出現藍色指紋圖示來提醒你這個網站已經加入了指紋密碼, 只需要掃描指紋, 剩下的工作就讓數位人科技來幫你完成.

保護你的個人電腦

使用指紋來登入微軟視窗作業系統. 可確保只有本人可以登入此電腦, 並且存取檔案及文件. 也不用有萬一電腦遭竊或遺失時, 有資訊安全的顧慮.

在視窗登入的畫面, 只要掃描指紋便可登入視窗. 如果其他人的視窗帳號正在使用中, 只要掃描指紋, 便可以立刻切換到個人的視窗帳號.

保護個人隱私文件

使用 DigitalPersona Personal 來保護你的機密資料. 像是財務紀錄, 工作檔案, 以及書面文件的備份電子檔案.

有了數位人科技的檔案保護機制, 檔案加密只需要短短幾秒鐘的時間. 在檔案上按滑鼠右鍵, 點選 "Encrypt Files" (檔案加密), 然後掃描指紋.

如果對一個資料夾加密, 檔案保護機制會套用到資料夾裡的所有檔案與子資料夾.

存取加密的檔案或資料夾時, 先點選檔案, 接著點滑鼠右鍵選擇 "Decrypt Files" (檔案解密) 然後掃描指紋即可.

安全地遨遊網路世界

不論是在哪裡上網, 或是透過怎樣的網路連接, COMODO TrustConnect 能在上網的時候, 保護個人隱私以及與資料傳輸的安全性.

COMODO TrustConnect 是 DigitalPersona Personal 的一個選擇性的附件, 讓使用者不管是在家裡, 機場, 或是圖書館以及任何地方, 進行網際網路連線時, 資訊的安全性都能受到保障.
