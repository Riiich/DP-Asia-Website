用指紋辨識保護所有隱私安全

DigitalPersona Personal 是適合個人以及家庭使用的套裝軟體. 它以指紋辨識來取代目前的密碼, 從此不用再擔心忘記密碼了.

只需要手指輕輕的一按, 就可以讓你:

  • 避免非本人不當地登入電腦並取得資料.
  • 登入電子郵件, 線上金融業務, 還有其他常用的網站及網路應用的技術.
  • 保護機密文件, 例如財務資料與個人隱私文件.
