One Touch I.D. 軟體開發工具

高效能指紋辨識能夠達到快速觸碰即可驗證.

  • 快速的辨識 — 只要簡單一個觸碰即可快速又準確的驗證
  • 高擴充性 — 在上千萬個使用者群體當中, 迅速準確的對應到單一個人
  • 強化安全性 — 消除詐欺
  • 減低IT成本 — 不需要提供使用者名稱, 密碼, 身分證號碼, 或刷卡
  • 增加使用者生產力 — 不需要記憶使用者名稱, 密碼, 身分證號碼, 或刷卡
  • 快速有效率的研發 — .NET 程式界面讓應用軟體研發更容易
  • 經濟實惠的授權方案 — 不需要每個使用者或每個模組的授權費

數位人科技所提供的One Touch I.D. 軟體開發工具(SDK)讓開發者研發他們的指紋辨識軟體能夠有快速, 有效率以及簡單的特性. One Touch I.D. SDK 延續了數位人科技的優良傳統, 為了開發指的關鍵性應用提供了穩定且可靠的開發工具. One Touch I.D. SDK 延伸了數位人科技軟體開發工具到其他平台像是One Touch for Windows, One Touch for Linux, 以及Gold CE.

用數位人科技One Touch for Windows工具包所開發出來的視窗應用軟體能夠改善指紋辨識的識別能力. 適用Windows XP與Vista的One Touch I.D. SDK完全免費 - 並不需要額外的個人授權費用. 一旦指紋被登錄, 它能夠儲存在任何資料庫或電腦當中, 並且從世界各地存取.

One Touch I.D. 2.1.1 新增特色

One Touch ID 2.1.1 包含了許多客戶期盼的新功能. 包括:

  • 支援64位元的作業系統. 提供內建更大的I.D.名單
  • 允許建立超過20,000個模組的I.D.名單
  • 支援單一手指與兩個手指的指紋辨識
  • 把指紋辨識擴及到模組化, 讓開發人員能夠執行雙重驗證
  • 包含了One Touch for Windows 1.4.0.1 RTE版本. 在執行之前不需要再重複編譯
  • 也同時包含支援Windows .NET的文件跟範例程式碼
