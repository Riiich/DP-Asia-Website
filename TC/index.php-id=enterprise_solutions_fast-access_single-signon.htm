Fast Access and Single SignOn

辦公室裡再也不需要密碼

使用者跟密碼往往是對立的。企業如果施行強力密碼的政策，或是需要不同的密碼來登入各種商務應用系統時，常常會發生，使用者為了少記幾個密碼，或是少打幾個字，使用一些不合乎安全要求的手段。例如，不管進入哪個系統都只用相同的一組密碼，寫在便利貼上，或是大家共用密碼等等。 而且常常需要協助以解決登入的問題。

更加方便而且又兼顧安全性

使用數位人科技的 DigitalPersona Pro 產品內含的單一登入 Single SignOn 功能，企業除了可以強化身分認證與資訊安全的策略，密碼同時不會再是使用者的噩夢了。

藉由指紋的技術，數位人科技把使用者的身分識別變得更容易了。既不需要其他的設備或是訓練，只要將自己的手指放上指紋掃瞄器，使用者馬上就可以使用這個新方案。

同時，因為使用者再也不需要在乎 - 需要多長的密碼，或是多久需要更換密碼，企業也因此能夠執行更強的身分驗證政策。結果就是，業務既能夠如願地安全運作，而且使用者也有了簡單又快速登入系統的方法。
