U.are.U 4500 指紋模組

允許開發者把指紋掃描器整合到它們自己的硬體當中.

U.are.U 4500 指紋模組是為何能夠整合到需要指紋驗證的OEM設備所設計的小型指紋掃描器. 它跟數位人科技的開發工具包 SDKs 能夠完美結合.

使用者只需要簡單的掃描手指, 掃描器便會自動且快速的擷取影像, 並且加密之後傳送到數位人科技的IDentity引擎做驗證.

數位人科技利用先進的光學指紋掃描技術, 來增進指紋影像的品質跟穩定性. U.are.U 4500指紋掃描器與IDentity引擎的結合提供了一套無與倫比的辨識能力來讀取最難辨識的指紋與環境.

4500HD — 高耐用度的模組

4500HD 是特別設計為了能夠在重要商業的應用上運作, 特別會要求指紋掃描器具備有高等級的耐用程度, 同時也能維持精準的辨識效能. 當被佈署在共享環境時, 指紋掃描器往往會遭到過度的使用與磨損, 如此會大大的降低辨識的效能與品質, 甚至設備的毀壞. 為了解決這個問題, 4500HD掃描器上有一層強硬且厚實的塗料來抵抗化學與物理上的磨損, 同時也能提供精準的判讀效能.

U.are.U 4500 與 4500HD 指紋模組, 適用於數位人科技全系列 軟體開發工具 SDKs.
