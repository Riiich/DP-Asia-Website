DigitalPersona Pro

DigitalPersona Pro 是中央管理的軟體，它允許您管理和你的小企業和企業之間執行嚴格的身份驗證.

你可以使用 DigitalPersona Pro 做到下列行為:

保護電腦的存取

有了DigitalPersona Pro, 您可以通過只有經過授權的使用者, 才可以登錄到可擕式電腦和桌面電腦來保護公司電腦的存取.

當DigitalPersona Pro被安裝到電腦上, 使用者需要利用指紋來驗證身分進而登入視窗系統. 這確保只有透過身分驗證才能存取電腦, 同時也能保護電腦上重要的商業資料, 且也保護企業免於資料外洩或遺失的風險.

安全地登入企業應用

DigitalPersona Pro 為所有企業軟體與應用提供了一套類似單一登入的機制.

IT管理部門能夠輕易地新增程式或應用到應用登入名單當中. 而從那時候開始, 使用者再也不需要使用密碼登入.

降低電話求助的情況

有了單一登入的機制, DigitalPersona Pro 能夠明顯的減少你桌上的求助電話. 藉著消除所有跟登入電腦或程式有關的問題, DigitalPersona Pro 能夠讓企業開始減少支出.

DigitalPersona Pro 元件 

DigitalPersona Pro 包含了伺服器與用戶端軟體

DigitalPersona AD 伺服器

伺服器元件連接 Active Directory 和您的 IT 部門所提供的管理工具，允許集中管理和配置在整個組織內的用戶端軟體.

另外, Pro AD 伺服器能夠當作身分驗證伺服器來使用. 當使用者掃描其在他們電腦上的指紋時，伺服器完成指紋匹配過程, 並驗證使用者的身份. 如此可用來:

  • 藉著確保驗證過程是被保護的, 不受惡意軟體的攻擊, 來達到最大的安全等級;

  • 漫遊功能為使用者提供身份驗證過程應用的安全性. 每當使用者需要登錄到一台電腦上，他將只需掃描他的指紋並登錄而無需重新註冊他在所有電腦上的指紋.

 

DigitalPersona Pro 用戶端軟體

DigitalPersona Pro 用戶端軟體為使用者在登入視窗或應用時, 提供了驗證功能. DigitalPersona Pro 支援兩種用戶端設定:

  • DigitalPersona Pro Workstation 是選擇性的用戶端軟體, 如果組織當中每個使用者都有一個視窗帳戶時, 可以選用.
  • DigitalPersona Pro Kiosk 是用戶端軟體, 如果組織當中一些人共用一個視窗帳戶時, 可以選用.

數位人科技支援任何Workstationa與Kiosk混合搭配的環境.

支援的指紋掃描器

DigitalPersona Pro 支援絕大多數在市場上的指紋掃描器.

筆記型電腦內鍵感應器
 

在全球銷售幾千萬台筆型電腦已經內建了指紋掃描器. 所有主要的地腦製造商都為他們的商務電腦配上指紋掃描器. DigitalPersona Pro 與市場上的領導夥伴合作, 幾乎支援所有筆記型電腦上最好的指紋掃描器.

指紋掃描器周邊設備
 

如果你的公司不使用筆記型電腦, 或是桌上型電腦需要額外連接指紋掃描器, 數位人科技提供的高階使用者外接式指紋掃描器. 易於使用而且符合人體工學, U.are.U 4500 指紋掃描器提供了卓越的性能與可靠性.
