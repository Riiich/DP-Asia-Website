DigitalPersona Pro

DigitalPersona Pro 是中央管理的软体，它允许您管理和你的小企业和企业之间执行严格的身份验证.

你可以使用 DigitalPersona Pro 做到下列行为:

保护电脑的存取

有了DigitalPersona Pro, 您可以通过只有经过授权的使用者, 才可以登录到可擕式电脑和桌面电脑来保护公司电脑的存取.

当DigitalPersona Pro被安装到电脑上, 使用者需要利用指纹来验证身分进而登入视窗系统. 这确保只有透过身分验证才能存取电脑, 同时也能保护电脑上重要的商业资料, 且也保护企业免于资料外泄或遗失的风险.

更多详情 »

安全地登入企业应用

DigitalPersona Pro 为所有企业软体与应用提供了一套类似单一登入的机制.

IT管理部门能够轻易地新增程式或应用到应用登入名单当中. 而从那时候开始, 使用者再也不需要使用密码登入.

降低电话求助的情况

有了单一登入的机制, DigitalPersona Pro 能够明显的减少你桌上的求助电话. 藉着消除所有跟登入电脑或程式有关的问题, DigitalPersona Pro 能够让企业开始减少支出.

DigitalPersona Pro 元件 

DigitalPersona Pro 包含了伺服器与用户端软体

DigitalPersona AD 伺服器

伺服器元件连接 Active Directory 和您的 IT 部门所提供的管理工具，允许集中管理和配置在整个组织内的用户端软体.

另外, Pro AD 伺服器能够当作身分验证伺服器来使用. 当使用者扫描其在他们电脑上的指纹时，伺服器完成指纹匹配过程, 并验证使用者的身份. 如此可用来:

  • 藉着确保验证过程是被保护的, 不受恶意软体的攻击, 来达到最大的安全等级;

  • 漫游功能为使用者提供身份验证过程应用的安全性. 每当使用者需要登录到一台电脑上，他将只需扫描他的指纹并登录而无需重新注册他在所有电脑上的指纹.

 

DigitalPersona Pro 用户端软体

DigitalPersona Pro 用户端软体为使用者在登入视窗或应用时, 提供了验证功能. DigitalPersona Pro 支援两种用户端设定:

  • DigitalPersona Pro Workstation 是选择性的用户端软体, 如果组织当中每个使用者都有一个视窗帐户时, 可以选用.
  • DigitalPersona Pro Kiosk 是用户端软体, 如果组织当中一些人共用一个视窗帐户时, 可以选用.

数位人科技支援任何Workstationa与Kiosk混合搭配的环境..

支援的指纹扫描器

笔记型电脑内键感应器
 

在全球销售几千万台笔型电脑已经内建了指纹扫描器. 所有主要的地脑制造商都为他们的商务电脑配上指纹扫描器. DigitalPersona Pro 与市场上的领导夥伴合作, 几乎支援所有笔记型电脑上最好的指纹扫描器.

指纹扫描器周边设备
 

如果你的公司不使用笔记型电脑, 或是桌上型电脑需要额外连接指纹扫描器, 数位人科技提供的高阶使用者外接式指纹扫描器. 易于使用而且符合人体工学, U.are.U 4500 指纹扫描器提供了卓越的性能与可靠性.
