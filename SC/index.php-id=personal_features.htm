DigitalPersona Personal 可以帮你做些甚么?

DigitalPersona Personal 是一套使用指纹的生物特征，来进行身分认证的软体，以避免冒用身分的问题发生。 有了DigitalPersona Personal，让使用电脑与网路变得更加安全且容易.

DigitalPersona 有下列的优点 :

取代密码

使用指纹, 快速且安全地登入电子邮件, 线上金融业务, 以及常用的网站与网路应用的技术, 例如 Facebook, Skype, 或是网路银行帐号 !!

如果网页上有出现指纹辨识的图示, 表示 DigitalPersona Personal 可以加入这个网. 只需要输入一次帐号与密码, 再扫描指纹及登入之后, 以后就可以用指纹来登入该网页了.

 

只需要输入一次帐号与密码, 之后即可以用指纹来取代.

下一次登入网站时, 出现蓝色指纹图示来提醒你这个网站已经加入了指纹密码, 只需要扫描指纹, 剩下的工作就让数位人科技来帮你完成.

保护你的个人电脑

使用指纹来登入微软视窗作业系统. 可确保只有本人可以登入此电脑, 并且存取档案及文件. 也不用有万一电脑遭窃或遗失时, 有资讯安全的顾虑.

在视窗登入的画面, 只要扫描指纹便可登入视窗. 如果其他人的视窗帐号正在使用中, 只要扫描指纹, 便可以立刻切换到个人的视窗帐号.

保护个人隐私文件

使用 DigitalPersona Personal 来保护你的机密资料. 像是财务纪录, 工作档案, 以及书面文件的备份电子档案.

有了数位人科技的档案保护机制, 档案加密只需要短短几秒钟的时间. 在档案上按滑鼠右键, 点选 "Encrypt Files" (档案加密), 然后扫描指纹.

如果对一个资料夹加密, 档案保护机制会套用到资料夹里的所有档案与子资料夹.

存取加密的档案或资料夹时, 先点选档案, 接着点滑鼠右键选择 "Decrypt Files" (档案解密) 然后扫描指纹即可.

安全地遨游网路世界

不论是在哪里上网, 或是透过怎样的网路连接, COMODO TrustConnect 能在上网的时候, 保护个人隐私以及与资料传输的安全性.

COMODO TrustConnect 是 DigitalPersona Personal 的一个选择性的附件, 让使用者不管是在家里, 机场, 或是图书馆以及任何地方, 进行网际网路连线时, 资讯的安全性都能受到保障.
