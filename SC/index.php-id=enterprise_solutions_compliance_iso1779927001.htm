ISO 17799/27001

The ISO 17799/27001 framework is a comprehensive set of controls comprising best practices in information security. It has emerged as the most popular framework for establishing or improving an organization's information security management program. It is organized into ten major sections including three identity security related sections: System Access Control, Physical and Environmental Security and Personnel Security.

Identity Compliance and ISO17799/27001

Managing identity compliance is fundamental to meeting ISO17799/27001 requirements. DigitalPersona's products are critical in complying with the following requirements from the standard

Strong Passwords

  • Establish a password management process.
  • Encourage users to protect passwords.
  • Set up a good password management system.
  • Use enforced paths from user terminals to services.
  • Manage access to application systems.
  • Restrict access at OS Level.

Secure Communications

  • Preserve confidentiality, integrity, and availability of information.
  • Protect mobile and teleworking assets.

Multi-Factor Authentication

  • Use cryptographic methods to authenticate remote users using strong protection.
  • Identify and authenticate all users: optionally use biometric authentication technologies.
