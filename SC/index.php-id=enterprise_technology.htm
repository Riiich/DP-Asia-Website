DigitalPersona 企业解决方案的技术

数位人科技企业解决方案将最先进的身分验证技术，与企业网路环境以及业务应用软体做紧密地整合。

快速进入与单一登入 Single SingOn

数位人科技一直是使用生物特征做为单一登入 Single SignOn 技术的领导者。 数位人科技开发的 SignOn 管理工具，能让IT部门对各种业务应用软体做事前的设定，消费者因此不需为了使用单一登入 SSO，而费心地自行做各种设定。 

数位人科技的 One Touch SignOn 元件可设定成自动训练已便了解网页以及商务应用的登入设计。使用在这个设定时，IT 部门能够快速地将数位人科技的企业解决方案，安装到新的各种应用之中。如果是更复杂的网站登入与客制化应用，使用具备充分弹性的手动设定，也能满足所有的应用需求。 One Touch SignOn 工具可支援以基于 JAVA 及终端机设计的各种应用。

一旦IT部门训练 One Touch SignOn 以支援某个应用方案的登入方式，接下来就可以藉着 Active Directory 的机制布署，完整地将此设定配置给分众或是全部的使用用户。当使用者下一次使用到这个应用方案时，就可以使用指纹来做登入动做了。

与 Active Directory 的整合

数位人科技的 DigitalPersona Pr通过微软 Microsoft 的认证，以扩充 Active Directory schema 的方式，来储存更多的使用者重要讯息。而且因为使用了 Active Directory 本身的使用者介面，所以并不会有新的操作需要再去学习。IT部门可以使用 Active Directory Group Policy Editior 来建立 GPO，然后根据特定的需求来制定软体使用方式。

数位人科技 DigitalPersona Pro 的设计是与 Windows 系统的安全服务做紧密的整合，用以保护所有的资料与通讯。因为Windows 系统会将所有运行的事件都会被记录到 Windows Event Log 事件日志，所以 DigitalPersona Pro 所产生的相关事件，也可使用一般的入侵侦测或是安全稽核工具来操作分析。

数位人科技 DigitalPersona Pro 利用 Active Directory 以及 DNS 的系统发布机制服务方式，来满足企业对未来扩充性的考量。当加入新的 DigitalPersona Pro 验证伺服器时，服务纪录就会传送到提供网路内用户连线资讯的 DNS 伺服器。如此当扩充系统或是系统发生异常(系统超载或是故障)时，系统管理者就不需去为每一个客户再更改设定，便能让所有的用户都能自动地取得服务更新资讯，转换到另一个服务的伺服器。

指纹的技术

从1997年以来，数位人科技一直是，在商业与个人用途使用指纹生物特征技术的领导厂商。数位人科技相信，指纹是强身分认证与易于使用间的完美结合。

数位人科技所引领的生物特征科技，是由Caltech 以及 MIT 的科学家们所设计，并且在全球已有一亿个使用者使用。由于拥有业界领先的准确率与可靠性，数位人科技的生物特征技术，将安全防护做到了最大化并且拥有最好的使用者体验。

当使用者将手指放到指纹扫瞄器时，数位人科技的软体会分析扫瞄所得影像并且取得当中的独特的特征点 minutiae，并且转换成为数学式，指纹模块，来呈现。 因为经过转换的过程，所以实际的指纹影像是不会被保存下来，如此也可以进一步地保护使用者生物特征资料的隐私权。
