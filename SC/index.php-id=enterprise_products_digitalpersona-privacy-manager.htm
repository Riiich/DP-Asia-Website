数位人科技 Privacy Manager Pro

最新加入的数位人科技应用的成员 DigitalPersona Privacy Pro, 是一套针对小型企业而推出集中管理的安全通讯方案.

你可以用数位人科技 Privacy Manager Pro来做什么？

有了数位人科技Privacy Manager Pro, 您可以在组织内或企业伙伴之间, 进行数位签章加密的电子邮件和档案传送.

电子邮件的安全通讯

使用数位人科技Privacy Manager Pro, 使用者可以进行具有数位签章加密的电子邮件传送给同事或业务伙伴. 这有助于防止在企业内部与外部保护机密资讯.

Privacy Manager Pro 是易于使用且让使用者都可以享受到它的便利性. 当他们想要安全地传送电子邮件时, 只要点选特定按钮接着扫描指纹之后, 数位人科技便能帮你处理之后的事务.
 

保护企业资料

数位人科技Privacy Manager Pro允许使用者安全地共享企业内的机密文件. 如此一来便能保护企业免于资料遗失的风险, 当中包括:

  • 遗失客户资料, 像是合约资讯.
  • 企业机密的外泄, 像是企业营运计画.
  • 严禁外泄且受保护的企业机密文件, 像是法律文件.

只要简单的几个按键, 使用者便能够把文件加密, 而且可以选择哪些受信任的名单是允许存取.
 

改善工作流程的效率

借着提供易于使用与设置的数位签章功能, 数位人科技Privacy Manager Pro使您可以提高文档管理, 以及文档审查工作流程的效率, 进而降低成本.

数位人科技Privacy Manager Pro使得签名文件与保存保密文件变得更加容易. 使用者只需要点选特定的按钮并且扫描指纹, 之后数位人科技会处理剩下的事务, 像是, 数位签章会自动地被附加到文件档案里头.
 

数位人科技 Privacy Manager Pro 如何运作？

尖端科技让事情变得更简单

数位人科技Privacy Manager Pro 利用数位认证来达到数位签章与文件加密的功能. 数位认证或称数位ID, 是一个存在于电脑中的安全性档案, 而且与使用该电脑的使用者做紧密的连结.

数位认证具有功能强大与灵活使用的特性, 因此能够是用在各种不同类型的数位验证与加密上面. 当其他供应商都提供了不易使用的方案, 数位人科技Privacy Manager Pro能够提供了卓越的使用者体验归因于:

  • 微软Outlook与Office软体的整合性
  • 利用指纹生物资讯来做身任验证是很容易达成的行为

集中管理且易于布署

数位人科技 Privacy Manager Pro 包含了许多的元件, 当中包含了联系 Active Directory 当中的配置与管理元件. 如此一来伺服器软体便能够透过组织来布署数位认证的签章. 当企业安装了DigitalPersona Pro, 数位人科技 Privacy Manager Pro 便能够利用它数位认证的结构来加强整个方案的安全性.
