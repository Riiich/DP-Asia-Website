U.are.U 4500 指紋模组

允许开发者把指纹扫描器整合到它们自己的硬体当中.

U.are.U 4500 指纹模组是为何能够整合到需要指纹验证的OEM设备所设计的小型指纹扫描器.它跟数位人科技的开发工具包 SDKs 能够完美结合.

使用者只需要简单的扫描手指, 扫描器便会自动且快速的撷取影像, 并且加密之后传送到数位人科技的IDentity引擎做验证.

数位人科技利用先进的光学指纹扫描技术, 来增进指纹影像的品质跟稳定性. U.are.U 4500指纹扫描器与IDentity引擎的结合提供了一套无与伦比的辨识能力来读取最难辨识的指纹与环境.

4500HD — 高耐用度的模组

4500HD 是特别设计为了能够在重要商业的应用上运作, 特别会要求指纹扫描器具备有高等级的耐用程度, 同时也能维持精准的辨识效能. 当被布署在共享环境时, 指纹扫描器往往会遭到过度的使用与磨损, 如此会大大的降低辨识的效能与品质, 甚至设备的毁坏. 为了解决这个问题, 4500HD扫描器上有一层强硬且厚实的涂料来抵抗化学与物理上的磨损,同时也能提供精准的判读效能.

U.are.U 4500 與 4500HD 指紋模组,适用于数位人科技全系列 软体开发工具 SDKs.
