U.are.U 4500 指纹键盘

把所有优点集合到一个时尚且符合人体工学的键盘上.

U.are.U 指纹键盘是一个内建指纹扫描器的 104个按键标准WINDOWS键盘.把所有指纹辨识的好处整合到键盘当中.它是为了下列数位人科技的产品所设计: 企业用户產品, 以及 软体开发工具包 SDKs. 这个整合性的键盘包含了所有指纹扫描器的特性,准确的验证,高耐用程度还有便利性,但对一般个人电脑使用者来说,可以不需要同时连结指纹扫描器与键盘两个USB装置.

在使用上, 只要很简单地把手指放在扫描器式窗上, 扫描器便会快速且自动地截取指纹特征的影像, 接着由数位人科技的IDentity引擎来做验证.

数位人科技利用先进的光学指纹扫描技术, 来增进指纹影像的品质跟稳定性. U.are.U 4500指纹扫描器与IDentity引擎的结合提供了一套无与伦比的辨识能力来读取最难辨识的指纹与环境.
