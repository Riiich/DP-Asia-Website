One Touch for Linux 软体开发工具

数位人科技One Touch for Linux软体开发工具包(SDK)DK) 给了专业开发者在Linux 2.4或2.6的核心系统应用上, 对指纹辨识安全系统有更多发挥的空间. One Touch for Linux支援数位人科技U.are.U指纹扫描器跟模组. 开发工具提供一套API来撷取指纹扫描器读取的影像档案, 特征提取, 还有指纹的比对, 都能在ANSI C与C++的编程环境. 同时也包含一个完全分散式数位人科技的及时引擎, 范例程式码, 还有详细的使用说明手册.

产品支援下列Linux 2.4或2.6核心平台:

  • Novel Linux Desktop 9 SP3, 核心版本 2.6.5–7.244-default/smp
  • Novell Linux Point of Service SP2 核心版本 2.6.5–7.201-SLRS
  • Novell Linux Point of Service SP3 核心版本 2.6.5–7.155.29-SLRS
  • SUSE Linux Enterprise Desktop 10 SP1 核心版本 2.6.16.46–0.12-default/smp
  • Slackware 11 核心版本 2.6.18
  • Slackware 11 核心版本 2.4.33.3
  • Slackware 12 核心版本 2.6.21.5
  • Red Hat Linux WS V3 核心版本 2.4.21–50.EL/smp

购买技术支援能够减轻最终解决方案跟应用的研发负担.

*核心模式的驱动程式原始码是提供给开发者以便移植在其他的2.6 Linux上执行. 由于各种版本的2.6 Linux作业系统, 数位人科技没办法提供技术支援让开发者重新编译驱动程式原始码.
