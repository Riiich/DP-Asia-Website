数位人科技 Gold CE 软体开发工具

数位人科技 Gold CE 软体开发工具 (SDK) 给了专业开发者在视窗应用上, 对指纹辨识安全系统有更多发挥的空间. One Touch for Windows SDK 支援数位人科技U.are.U 指纹扫描器, 模组, 与键盘. 这个开发工具包含了eMbedded Visual C++与IDentity Engine的范例程式码. 数位人科技Gold CE SDK 支援数位人科技U.are.U 指纹扫瞄器, 以及U.are.U 指纹模组(分开销售).
