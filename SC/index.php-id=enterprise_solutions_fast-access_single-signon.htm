Fast Access and Single SignOn

办公室里再也不需要密码

使用者跟密码往往是对立的。企业如果施行强力密码的政策，或是需要不同的密码来登入各种商务应用系统时，常常会发生，使用者为了少记几个密码，或是少打几个字，使用一些不合乎安全要求的手段。例如，不管进入哪个系统都只用相同的一组密码，写在便利贴上，或是大家共用密码等等。 而且常常需要协助以解决登入的问题。

更加方便而且又兼顾安全性

使用数位人科技的 DigitalPersona Pro 产品内含的单一登入 Single SignOn 功能，企业除了可以强化身分认证与资讯安全的策略，密码同时不会再是使用者的噩梦了。

藉由指纹的技术，数位人科技把使用者的身分识别变得更容易了。既不需要其他的设备或是训练，只要将自己的手指放上指纹扫瞄器，使用者马上就可以使用这个新方案。

同时，因为使用者再也不需要在乎 - 需要多长的密码，或是多久需要更换密码，企业也因此能够执行更强的身分验证政策。结果就是，业务既能够如愿地安全运作，而且使用者也有了简单又快速登入系统的方法。
